# Task 6: Interactive Dashboards

## Overview
This advanced module upgrades from static image generation to fully interactive web-based data visualizations. Using the Plotly library and a global economic indicators dataset, it demonstrates the creation of interactive, animated, and 3D data stories.

## Key Visualizations Created
* Interactive line and bar charts with hover-tooltips.
* Scatter plots scaling with population data.
* Geographic choropleth mapping for global metrics.
* 3D scatter plots allowing free-camera rotation.
* Time-series animations mapping economic progression from 1952 to 2007.

**Included Files:**
* Jupyter Notebook (`.ipynb`) containing the Plotly source code.
* `Interactive_Dashboard.html`: A compiled, fully interactive web dashboard containing all generated charts. (Download and open in any web browser to interact).
