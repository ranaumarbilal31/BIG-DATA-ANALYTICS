# Task 3: Numerical Computing with NumPy

## Overview
This task explores high-performance mathematical computing using NumPy. Utilizing a simulated weather dataset, it covers everything from basic array manipulation to linear algebra and statistical analysis.

## Key Skills Demonstrated
* Creation and manipulation of 1D, 2D, and 3D arrays.
* Array indexing, slicing, reshaping, and broadcasting.
* Applying statistical functions (mean, median, standard deviation, percentiles).
* Linear algebra operations (dot product and matrix multiplication).
* **Performance Benchmarking:** Demonstrating NumPy's computational speed advantage over standard Python lists.

**Included Files:**
* Jupyter Notebook (`.ipynb`) with complete source code.
* Screenshot of the performance comparison output.
